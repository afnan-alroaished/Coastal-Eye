# Coastal-Eye

## Overview
Coastal-Eye is an AI-powered maritime monitoring platform designed to enhance coastal security using unmanned aerial vehicles (UAVs), computer vision, GIS, and photogrammetry technologies.

## Objectives
- Monitor coastal areas in real time.
- Detect suspicious maritime activities.
- Improve situational awareness.
- Support smart surveillance operations.

## Technologies
- Artificial Intelligence
- UAV (Drone)
- Computer Vision
- GIS
- Photogrammetry
- Pix4D
- CesiumJS
- Python

## Features
- Real-time monitoring
- AI object detection
- 3D mapping
- Orthomosaic visualization
- Threat detection
- Smart surveillance dashboard
