# Coastal-Eye

## AI Maritime Monitoring Platform

Coastal-Eye is an AI-powered maritime monitoring platform developed to enhance coastal security and situational awareness through the integration of Unmanned Aerial Vehicles (UAVs), Artificial Intelligence, Computer Vision, GIS, and Photogrammetry technologies.

---

## Project Overview

This project demonstrates how modern AI and drone technologies can be combined to monitor coastal environments, detect potential threats, generate high-accuracy geospatial data, and support intelligent decision-making for maritime security.

---

## Objectives

- Monitor coastal areas in real time.
- Detect suspicious maritime activities.
- Improve maritime situational awareness.
- Generate accurate photogrammetry products.
- Support smart surveillance operations.

---

## Key Features

- AI-powered object detection
- UAV-based coastal monitoring
- 3D Reconstruction using Pix4D
- Orthomosaic generation
- GIS visualization
- Smart surveillance dashboard
- Maritime threat detection
